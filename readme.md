# 해당 파일들을 가지고 실행할 때 파일 실행 순서

#1단계 : 프로그램 시작 및 데이터베이스 연결 :: 초기 세팅

- 1. **라이브러리 로드:** `random`, `getpass`, `supabase` 모듈을 불러옵니다.
- 2. **로컬 데이터 공간 생성:** 내 컴퓨터 메모리에 점수를 임시 저장할 텅 빈 리스트 `ranking_data = []`를 만듭니다.
- 3. **🔑 Supabase 연결 실행:** `connect_supabase()` 함수가 자동으로 실행되면서 터미널창에 다음과 같이 뜹니다.
- 4. **연결 성공:** 올바르게 입력하면 `Supabase 연결 완료`가 뜨고 전역 변수 `supabase`에 연결 객체가 저장됩니다.

#2단계 : 메인 메뉴 대기 상태 :: 사용자의 입력을 기다리는 메뉴판 출력

#3단계 : 사용자의 선택에 따른 내부 작동 순서

- 1번을 눌렀을 때 ➡️ 퀴즈 게임 및 기록 저장

- 2번을 눌렀을 때 ➡️ 데이터 시각화 분석
  - 사용자가 번호를 선택하면 해당 조건문 내부에서 **Supabase 특정 테이블 조회 ➡️ 판다스 데이터프레임 변환 ➡️ Matplotlib/Seaborn 그래프 생성** 순서로 처리됩니다.
  - `plt.show()`가 실행되면서 VS Code 화면에 독립된 그래프 팝업창이 새로 뜹니다.
  - 뜬 그래프 창을 **마우스로 닫아야만** 다음 코드로 진행되며, 터미널에 다시 번호 입력창이 활성화됩니다.
  - `no`를 입력하면 서브 루프를 빠져나와 다시 2단계의 메인 메뉴판으로 돌아갑니다.

- 3번을 눌렀을 때 ➡️ 로컬 랭킹 보기

  - 프로그램을 켠 이후 현재 판에서 플레이했던 기록들만 모아서 최고 점수 순으로 정렬한 뒤 상위 5등까지 화면에 출력합니다.

- 4번을 눌렀을 때 ➡️ DB 랭킹 보기

  - 내 컴퓨터 기록이 아니라 **Supabase 서버의 `game_rankings` 테이블에 누적된 전체 유저의 데이터**를 원격으로 조회해옵니다. 점수가 높은 순, 시도 횟수가 적은 순으로 정렬하여 실시간 TOP 10을 터미널에 출력합니다.

- 5번을 눌렀을 때 ➡️ 게임 안전 종료

  - `break`문이 작동하면서 무한히 돌던 메인 `while` 루프가 깨지고 프로그램이 메모리에서 완전히 내려가며 종료됩니다.
